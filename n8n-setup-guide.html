1
Get your Google Gemini API Key
~5 mins
1
Go to Google AI Studio
Visit aistudio.google.com and sign in with your Google account.
2
Create an API key
Click Get API Key (left sidebar) → Create API key → select or create a Google Cloud project when prompted.
3
Copy and save the key
Your key starts with AIza... — copy it and save it somewhere safe (Notes, password manager).
✅ Gemini API has a generous free tier — generating training plans will likely cost you nothing at all.
2
Set up n8n
~5 mins
1
Create a free n8n Cloud account
Go to n8n.io → click Start for free → sign up. You'll land on your n8n dashboard.
2
Create a new workflow
Click + New Workflow in the top right. Name it "STRIDE Plan Generator".
3
Build the Workflow
~10 mins
1
Add a Webhook node
Click + on the canvas → search Webhook → select it.

In the settings panel set:
  • HTTP Method → POST
  • Path → type stride-plan
  • Respond → Using Respond to Webhook Node
2
Add a Google Gemini node
Click + after the Webhook → search Google Gemini → select it → choose Generate Text as the operation.

Click Create new credential:
  • Select API Key as the auth type
  • Paste your AIza... key → click Save
Then set:
  • Model → gemini-1.5-flash (fast & free) or gemini-1.5-pro (more detailed)
  • Prompt → click the {} expression icon and paste:
Paste this into the Prompt field
{{ $json.body.prompt }}
3
Add a "Respond to Webhook" node
Click + after Gemini → search Respond to Webhook → select it.

Set Respond With → JSON

In the Response Body field, click {} and paste:
Paste this into Response Body
{ "workouts": {{ $json.text }} }
💡 Gemini returns its response in $json.text — slightly different from OpenAI. STRIDE parses the JSON array inside automatically.
⚠️ If you get an error, open the Gemini node's output in the n8n execution log and check the exact field name — it may be $json.candidates[0].content.parts[0].text depending on your n8n version.
4
Get your Webhook URL
Click on the Webhook node. You'll see two URLs:

  • Test URL — only works while n8n is open and listening
  • Production URL — works 24/7 once the workflow is active ← use this one

Copy the Production URL. It looks like:
https://yourname.app.n8n.cloud/webhook/stride-plan
5
Activate the workflow
Click the Inactive toggle in the top right of n8n → flip it to Active. Your webhook is now live.
6
Paste the URL into STRIDE and test!
Go back to your STRIDE dashboard → paste the Production URL into the n8n Webhook URL field → fill in your running details → hit ⚡ Generate Plan.
🎉 If it works, your calendar will fill up with a full month of workouts in about 10–20 seconds!

Troubleshooting
Problem Fix
"Something went wrong" Open n8n → Executions (left sidebar) → click the failed run to see the error
Webhook not receiving data Make sure you're using the Production URL (not Test URL) and the workflow toggle is Active
Gemini error / 403 Check your API key starts with AIza... and is pasted correctly. Make sure the Gemini API is enabled in your Google Cloud project at console.cloud.google.com
Calendar shows empty Gemini may have wrapped the JSON in markdown code fences (```json). STRIDE strips these automatically, but check the Executions log for the raw response to confirm
CORS error in browser console Add a "Set Headers" node after the Webhook node. Add header: Access-Control-Allow-Origin: *
